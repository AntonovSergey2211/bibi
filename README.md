## Scripts

### [build]
Build all <br/>
`yarn build`

Build few packages <br/>
`yarn build geom events sample`

### [test]
Test all packages <br/>
`yarn test`

### [lint]
Linting all packages <br/>
`yarn lint`

### [clean]
Clean all packages <br/>
`yarn clean`
